# 1.5 主流PLC品牌介绍

> 本节将介绍市场上主流的PLC品牌和型号

## 学习目标

- 了解主流PLC品牌
- 掌握各品牌的特点和适用场景
- 能够根据需求选择合适的PLC

---

## 1. 西门子（Siemens）系列

### 1.1 品牌概述

西门子是德国工业巨头，PLC产品在全球市场占有率第一，尤其在欧洲和中国市场占据主导地位。

```
西门子PLC市场地位：

全球市场份额：约30%
主要市场：欧洲、中国、亚太
优势领域：汽车、制药、食品饮料、过程控制
```

### 1.2 主要产品系列

| 系列 | 定位 | I/O点数 | 特点 |
|------|------|---------|------|
| **LOGO!** | 微型 | <24点 | 智能继电器，简单控制 |
| **S7-200 SMART** | 小型 | 14-60点 | 经济型，入门首选 |
| **S7-1200** | 小型 | 14-284点 | 紧凑型，功能强大 |
| **S7-1500** | 中大型 | 可扩展 | 高性能，运动控制 |
| **S7-300** | 中型 | 可扩展 | 经典型号（已停产） |
| **S7-400** | 大型 | 可扩展 | 复杂系统（已停产） |

### 1.3 编程软件

```
西门子编程软件体系：

┌─────────────────────────────────────────────────────┐
│               TIA Portal (博途)                      │
│  ┌─────────────────────────────────────────────┐   │
│  │  集成开发环境，支持S7-1200/1500系列          │   │
│  │  功能：编程、组态、仿真、诊断                │   │
│  └─────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────┤
│  STEP 7 MicroWIN ─── S7-200系列（已停产）          │
│  STEP 7 V5.x    ─── S7-300/400系列                 │
│  LOGO! Soft     ─── LOGO!系列                      │
└─────────────────────────────────────────────────────┘
```

### 1.4 西门子PLC特点

**优势：**
- ✅ 产品线完整，从微型到大型全覆盖
- ✅ 软件功能强大，开发效率高
- ✅ 可靠性高，适合严苛工业环境
- ✅ 技术支持和培训资源丰富
- ✅ 与西门子其他产品（变频器、HMI）无缝集成

**劣势：**
- ❌ 价格相对较高
- ❌ 软件学习曲线较陡
- ❌ 部分功能需要额外授权

---

## 2. 三菱（Mitsubishi）系列

### 2.1 品牌概述

三菱电机是日本自动化领域的龙头企业，PLC产品在亚洲市场份额领先，以性价比高、易学易用著称。

```
三菱PLC市场地位：

全球市场份额：约15%
主要市场：日本、中国、东南亚
优势领域：电子、半导体、包装机械、纺织
```

### 2.2 主要产品系列

| 系列 | 定位 | I/O点数 | 特点 |
|------|------|---------|------|
| **FX5U** | 小型 | 32-256点 | 最新一代，高性能 |
| **FX3U** | 小型 | 16-256点 | 经典型号，市场保有量大 |
| **FX3G** | 小型 | 14-60点 | 经济型，入门推荐 |
| **FX3S** | 微型 | 10-30点 | 超紧凑，简单控制 |
| **iQ-R系列** | 中大型 | 可扩展 | 高端，多CPU架构 |
| **Q系列** | 中型 | 可扩展 | 模块化，灵活配置 |
| **L系列** | 中型 | 可扩展 | 紧凑型模块化 |

### 2.3 编程软件

```
三菱编程软件体系：

┌─────────────────────────────────────────────────────┐
│              GX Works3                              │
│  ┌─────────────────────────────────────────────┐   │
│  │  支持FX5U、iQ-R、iQ-F等新型号               │   │
│  │  全新界面，结构化编程                        │   │
│  └─────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────┤
│  GX Works2  ─── FX3U/3G/3S、Q系列、L系列           │
│  GX Developer ─── 老型号（已淘汰）                 │
└─────────────────────────────────────────────────────┘
```

### 2.4 三菱PLC特点

**优势：**
- ✅ 性价比高，适合成本敏感项目
- ✅ 编程简单直观，学习门槛低
- ✅ 高速计数、定位功能内置
- ✅ 在亚洲有广泛的技术支持网络
- ✅ 市场保有量大，配件容易获取

**劣势：**
- ❌ 大型系统方案不如西门子成熟
- ❌ 软件功能相对简单
- ❌ 国际化支持不如欧美品牌

---

## 3. 欧姆龙（Omron）系列

### 3.1 品牌概述

欧姆龙是日本知名自动化企业，以传感器和控制器闻名，PLC产品在机械设备制造领域应用广泛。

### 3.2 主要产品系列

| 系列 | 定位 | 特点 |
|------|------|------|
| **NX1P2** | 小型 | 内置EtherCAT，运动控制 |
| **NJ系列** | 中大型 | 机器自动化控制器 |
| **CP1H** | 小型 | 高速处理，定位功能 |
| **CP1L** | 小型 | 经济型，入门级 |
| **CP1E** | 微型 | 超小型，简单控制 |
| **CJ2系列** | 中型 | 模块化，高性能 |

### 3.3 欧姆龙PLC特点

**优势：**
- ✅ 与欧姆龙传感器、伺服完美配合
- ✅ EtherCAT运动控制方案成熟
- ✅ 软件界面友好
- ✅ 产品可靠性高

**劣势：**
- ❌ 市场份额相对较小
- ❌ 价格中等偏上

---

## 4. 罗克韦尔/AB（Allen-Bradley）系列

### 4.1 品牌概述

罗克韦尔自动化（Rockwell Automation）是美国最大的自动化公司，Allen-Bradley（AB）是其PLC品牌，在北美市场占据主导地位。

```
AB PLC市场地位：

全球市场份额：约20%
主要市场：北美、南美
优势领域：汽车、石油、采矿、重工业
```

### 4.2 主要产品系列

| 系列 | 定位 | 特点 |
|------|------|------|
| **Micro800** | 微型 | 紧凑经济 |
| **Micro850** | 小型 | 独立控制 |
| **CompactLogix** | 中型 | 模块化，EtherNet/IP |
| **ControlLogix** | 大型 | 高性能，复杂系统 |
| **SLC500** | 中型 | 经典型号（已停产） |
| **PLC-5** | 大型 | 老型号（已停产） |

### 4.3 编程软件

```
罗克韦尔编程软件：

┌─────────────────────────────────────────────────────┐
│           Studio 5000 Logix Designer                │
│  ┌─────────────────────────────────────────────┐   │
│  │  支持CompactLogix、ControlLogix系列          │   │
│  │  统一编程环境                                │   │
│  └─────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────┤
│  Connected Components Workbench ─── Micro800系列   │
│  RSLogix 500  ─── SLC500、MicroLogix               │
│  RSLogix 5    ─── PLC-5                            │
└─────────────────────────────────────────────────────┘
```

### 4.4 AB PLC特点

**优势：**
- ✅ 在北美市场认可度高
- ✅ 大型复杂系统方案成熟
- ✅ EtherNet/IP通信标准
- ✅ 与罗克韦尔全线产品集成

**劣势：**
- ❌ 价格较高
- ❌ 在亚洲市场份额较小
- ❌ 中国技术支持资源相对有限

---

## 5. 施耐德（Schneider）系列

### 5.1 品牌概述

施耐德电气是法国跨国企业，通过收购莫迪康（Modicon）进入PLC领域，Modbus协议就是其发明的。

### 5.2 主要产品系列

| 系列 | 定位 | 特点 |
|------|------|------|
| **M221** | 微型 | 紧凑经济，简单控制 |
| **M241** | 小型 | 运动控制，高性能 |
| **M251** | 小型 | 以太网功能强大 |
| **M340** | 中型 | 模块化，过程控制 |
| **M580** | 大型 | 高可用性，冗余 |
| **Quantum** | 大型 | 过程控制（经典） |

### 5.3 施耐德PLC特点

**优势：**
- ✅ Modbus协议原生支持
- ✅ 过程控制领域经验丰富
- ✅ 与施耐德电气产品整合
- ✅ 性价比较高

**劣势：**
- ❌ 在中国市场份额相对较小
- ❌ 产品线整合后略显复杂

---

```

├─────────────────────────────────────────────────────┤
│  汇川技术  │  深圳    │  H系列、Easy系列 │  综合实力最强│
│  台达电子  │  台湾    │  DVP、AS系列     │  性价比高   │
│  信捷电气  │  无锡    │  XC系列          │  小型PLC专业│
│  和利时    │  北京    │  LK/LM系列       │  过程控制   │

```

### 6.2 汇川技术

```
汇川PLC产品线：
```

### 6.3 台达电子

| 系列 | 定位 | 特点 |
|------|------|------|
| **AS系列** | 中小型 | 最新一代，功能强大 |
| **AH500** | 中型 | 模块化设计 |
| **DVP系列** | 小型 | 经典型号，市场保有量大 |

### 6.4 国产PLC特点

**优势：**
- ✅ 价格优势明显（进口的1/3-1/2）
- ✅ 本土化服务响应快
- ✅ 部分品牌技术水平已接近国际品牌
- ✅ 支持国产化要求的项目

**劣势：**
- ❌ 品牌认知度待提高
- ❌ 大型复杂系统经验不足
- ❌ 部分品牌可靠性有待验证

### 7.2 各品牌适用场景推荐

| 场景 | 推荐品牌 | 推荐型号 |
|------|----------|----------|
| **学习入门** | 西门子、三菱 | S7-200 SMART、FX3G |
| **小型设备** | 三菱、信捷 | FX5U、XC系列 |
| **生产线** | 西门子、三菱 | S7-1200/1500、Q系列 |
| **汽车行业** | 西门子、AB | S7-1500、ControlLogix |
| **北美项目** | AB | CompactLogix |
| **成本敏感** | 国产品牌 | 汇川、台达 |
| **过程控制** | 西门子、施耐德 | S7-400、M580 |
| **运动控制** | 欧姆龙、汇川 | NJ系列、AM600 |

### 7.3 初学者建议

```
初学者学习路线建议：

【推荐1】西门子路线
  S7-200 SMART (入门) → S7-1200 (进阶) → S7-1500 (高级)
  优点：就业面广，资料多

【推荐2】三菱路线
  FX3G (入门) → FX3U/FX5U (进阶) → Q系列 (高级)
  优点：易学，设备制造行业需求大

【推荐3】国产路线
  信捷XC (入门) → 汇川H3U (进阶) → 汇川AM600 (高级)
  优点：性价比高，未来发展潜力大
```
